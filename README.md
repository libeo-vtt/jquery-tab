# jQuery Tab Plugin
Accessible, responsive and configurable jQuery tab plugin

## Usage

1. Include jQuery:

	```html
	<script src="path/to/jquery.min.js"></script>
	```

2. Include plugin's code:

	```html
	<script src="path/to/jquery.tab.js"></script>
	```

3. Call the plugin:

	```javascript
	$('#element').tab({
		// config
	});
	```
